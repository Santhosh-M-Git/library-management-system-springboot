# library-management-system-springboot
REST-based Library Management System using Spring Boot

# Library Management System

## Technologies Used
- Java
- Spring Boot
- REST API
- Oracle SQL

## Features
- Add Book
- View Books
- Update Book
- Delete Book
- User Registration & Login

## Database
Oracle SQL

## API Endpoints
- POST   /api/books
- GET    /api/books
- GET    /api/books/{id}
- PUT    /api/books/{id}
- DELETE /api/books/{id}

## How to Run
1. Clone the repository
2. Import into STS
3. Run as Spring Boot Application
